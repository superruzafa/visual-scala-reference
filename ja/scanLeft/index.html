    scanLeft

    trait Collection[A] {
  def scanLeft[B](z: B)(op: (B, A) => B): Collection[B]
}

    scanLeft は、二項演算子 op をこのコレクションの要素に左から右へ適用していく中間結果のコレクションを作成します。 最初に op が適用されるときには、初期値 z が入力されます。

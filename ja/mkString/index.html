    mkString

    trait Collection[A] {
    def mkString: String
    def mkString(sep: String): String
    def mkString(start: String, sep: String, end: String): String
}

    mkString は、このコレクションの各要素の文字列表現を結合した String を作り上げます。 各部分の間にはさまる sep、開始と終了の文字列としてそれぞれ start と end を、任意で使えます。

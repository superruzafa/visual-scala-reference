aggregate

trait Collection[A] {
  def aggregate[B](z: => B)(seqop: (B, A) => B, combop: (B, B) => B): B
}

aggregate aplica el operador binario sepop a porciones de elementos y luego combina los resultados intermedios con el operador binario combop. El valor z es el elemento neutro para el tipo B y puede ser utilizado múltiples veces.
